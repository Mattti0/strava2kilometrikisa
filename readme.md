# stava2kilometrikisa

An application to fill the gap between Strava and Kilometrikisa.

## Installation

1. clone this project
2. npm run build
3. Visit your localhost/dev environment on port 3000
3. enjoy.

## Usage

The default dev port for this node application is 3000.
